в якості локального сервера був використаний сервіс MAMP через його сучасність та простоту у налаштуванні. При відкритті MAMP з'являються кнопки для запуску сервера та для його конфігурації. Для запуску або зупинки використовуюємо відповідну кнопку "Start/stop server"

Можливі налаштування MAMP: start/stop, ports, version php, де буде знаходитись наш сервер"web server",підключити mysql,завантажити до cloud

Також для налаштування можливе використання файлу httpd.conf

в цьому файлі ми бачимо конфігурації нашого віртуального хоста

для того щоб робити якісь зміни в сервері потрібно розкоментувати наступні строки

Також для налаштування можливе використання файлу httpd-vhosts.conf для змiни хосту:

В данному файлі ми прописуємо: сервер, де буде знаходитись директорія, назва серверу, назву сервера аліансу та можливі помилки.
